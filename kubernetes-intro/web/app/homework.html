Моя домашняя работа

Для выполнения домашней работы нужно создать Dockerfile, в котором будет описан образ:

  1. Запускающий web-сервер на порту 8000 (можно использовать любой способ)
  2. Отдающий содержимое директории /app внутри контейнера (например, если в директории /app лежит файл homework.html , то при запуске контейнера данный файл должен быть доступен по URL http://localhost:8000/homework.html )
  3. Работающий с UID 1001
  4. После того, как Dockerfile будет готов: В корне репозитория создайте директорию kubernetes-intro/web и поместите туда готовый Dockerfile. Соберите из Dockerfile образ контейнера и поместите его в публичный Container Registry (например, Docker Hub)
